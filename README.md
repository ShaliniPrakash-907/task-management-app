# TaskFlow - Task Management Application

## Overview
TaskFlow is a full-stack task management web application developed using PHP and MySQL.

## Features
- User Registration & Login
- Task Creation
- Task Editing
- Task Deletion
- Search Tasks
- Filter Tasks by Status and Priority
- Dashboard Analytics
- Due Date Tracking
- Responsive Design

## Tech Stack
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Project Structure
- Home Page
- Dashboard
- Add Task
- Edit Task
- Delete Task
- User Authentication
